💾 DATABASE SCHEMA (Simplified Identity)

Users Table:
├── Id                   (Primary Key)
├── UserName            (Login name)
├── Email               (User email)
├── Password            (PBKDF2 hash - renamed from PasswordHash)
├── FullName            (Display name - custom)
├── IsActive            (Enabled/disabled - renamed from LockoutEnabled)
├── AccessFailedCount   (Failed login attempts)
└── SecurityStamp       (Token invalidation)

Roles Table:
├── Id                   (Primary Key)
└── Name                (Role name: Admin, Staff)

Benefits: Faster, cleaner, easier to maintain!
